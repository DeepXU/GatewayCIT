# Cryptography Tools
Below is a list of useful tools for completing the cryptography challenges in the NCL.

- [ASCII Converter](https://www.branah.com/ascii-converter)
- [Base64 Decode and Encode](https://www.base64decode.org/)
- [CyberChef](https://gchq.github.io/CyberChef/)
- [Hex to ASCII text converter](http://www.rapidtables.com/convert/number/hex-to-ascii.htm)
- [rot13.com](http://rot13.com/)
- [Wingdings Translator](https://lingojam.com/WingdingsTranslator)
